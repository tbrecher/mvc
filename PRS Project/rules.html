Rock, Paper, Scissors!!!

Rules

The game of Rock, Paper, Scissors is based on trust, skill and determination. Each player must:
  • Each player will choose either Rock, Paper or Scissors.
  • Rock beats Scissors.
  • Scissors beats Paper.
  • Paper beats Rock.
  • If there is a tie, the game is repeated until there is a winner.
